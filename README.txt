SPACE est la touche d'action.
Z est pour annulé un choix (utile en combat surtout).
Près d'un PNJ, il faut faire ESCAPE pour lui parler.
L pour ouvrir le menu en jeu.
F12 pour ouvrir le menu de débogage
Utiliser les touches fléchées (ou ASDW) pour progresser.
